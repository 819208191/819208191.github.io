一、平台业务总览

业务 1：GPT Plus/Pro 网页拼车服务

适用人群：学生、办公、科研轻量聊天

号池说明：自建 Team/Plus/Pro 订阅账号池，账号隔离轮换风控，低封号风险。

20 元包月规则限制：GPT5.4 不限对话，GPT5.5 每日 30 次，单次输出上限 1500 字。每日晚间 20-24 点为访问高峰期，可能会触发限流排队。
违规规则：严禁利用本通道运行爬虫或进行批量文本生成。一经检测违规，直接关停套餐且不予退款。

业务 2：Token 中转 API 服务（开发 / 企业主力）

双渠道货源：自建订阅逆向 API + Namax 上游官方 API，模型全面支持 gpt-5.4 和 gpt-5.5。

三档分组：低价测试池（限流）、标准商用池、企业专线池。

优势：国内环境直连无需魔法，无缝兼容 NewAPI、Cursor、CherryStudio 等主流商用及开源开发工具。

功能特色：支持独立用户令牌分发、自定义日/月 Token 限额、输入与输出分开独立计费。大客户及代理商可享受阶梯折扣。

业务 3：LangBot 龙虾一站式部署服务

服务包含：包含服务器基础环境配置、Docker 环境安装、源码编译部署、域名 HTTPS 证书申请、对接本站 Token 渠道以及限流规则配置。

适用场景：搭建个人私有 AI 站点、工作室内部业务机器人。

售后保障：附赠独家运维教程，并提供基础技术问题长期答疑服务。

  • 充值档位：支持 10元 / 30元 / 50元 / 100元 固定面额充值。

三、网页拼车套餐使用教程

账号充值完毕后，在后台购买对应的包月套餐，系统将自动开通网页对话入口。

额度刷新规则：每日 0 点 准时重置 GPT5.5 的当日对话次数。

注意：高峰期排队、响应延迟属于正常的低价通道现象。重度使用和追求极速响应的用户，强烈建议切换至 API 按量付费服务。

四、NewAPI 上游对接教程（Namax 渠道）

如需使用 NewAPI 进行上游对接，请严格使用以下固定参数：

API URL 
https://api.namax.ai/v1
Support Models 
gpt-5.4, gpt-5.5

后台具体操作步骤：

  1. 登录 NewAPI 后台，依次点击 渠道管理 → 新增渠道。
  2. 在类型下拉菜单中选择 OpenAI。
  3. 在对应的接口地址栏填入上游地址 https://api.namax.ai/v1，并正确填入您的上游密钥。
  4. 在模型列表框中填入或勾选上述支持的模型，最后点击 测试连通性。

渠道绑定分组：建议将特价、商用、企业分组分开建立渠道绑定，以便灵活设置倍率控制分销成本。

令牌创建：在后台令牌页面新建令牌，将其绑定至对应的用户分组，并自定义并发数与每日 Token 消耗限额。

核心概念区分：上游 Namax 密钥 是您在平台内部的进货凭证；而用户令牌则是您对外分发给最终用户或应用调用的密钥。两者相互隔离，请勿混淆。

五、龙虾 LangBot 服务器一键部署教程

前置条件：准备一台 Linux 云服务器，并在安全组/防火墙中开放 80、443 以及 3000 端口。

请复制以下完整一键部署命令在您的服务器终端中运行：

bash 
apt update && apt upgrade -y
curl -fsSL https://get.docker.com | bash
systemctl enable docker && systemctl start docker
docker run -d --name langbot --restart always -p 3000:3000 -v /data/langbot:/data xxxx/langbot:latest

部署完成后，在浏览器中访问 服务器IP:3000 即可登录管理后台。

渠道配置：进入后台配置，填入本站的 NewAPI 域名以及您的个人用户令牌，并合理设置对话限流，避免额度出现意外超额消耗。

六、包月套餐限流规则说明

20元月卡硬性限制：GPT5.5 每日调用上限为 30 次，单次上下文输出 1500 字封顶。

额度耗尽处理：若包月额度耗尽，可在后台直接充值切换为按量计费调用，或一键升级至更高阶的商用包月套餐。

使用场景限制：此通道严禁用于任何批量脚本、网络爬虫等高并发、高消耗场景。

七、常见报错故障排查

  • 401 无权限 (Unauthorized)：请检查令牌是否复制完整、账号可用余额是否不足、或该令牌因违规已被系统自动封禁。
  • 404 接口错误 (Not Found)：通常原因为配置的 BaseURL 格式不规范，请确保末尾带有 /v1。
  • 429 限流 (Too Many Requests)：代表当日调用次数或 Token 额度已耗尽，或正处于晚间高峰期通道限流排队状态。
  • 调用超时 (Timeout)：低价测试通道受网络波动影响较大，如对稳定性有高要求，请直接切换至标准商用分组重试。
  • 充值未到账：偶发性收款挂机软件离线。请不要惊慌，直接向客服提供您的商户订单号，等待人工核对补发。
