API 上游对接

平台原生兼容主流开发生态（如 NewAPI、NextChat、Codex 等智能体）。通过简单的配置即可接入支持 API 的各类智能体。

[ 图 12. 渠道对接配置界面 ]

接入配置方法

接入支持 API 的智能体只需以下简单步骤：

  1. 在 令牌管理 中创建一个密钥
  2. 选择一个合适的 分组
  3. 在智能体配置中填入以下信息
// Base URL（统一使用我们的官网地址）
base_url: "https://masikeai.cc.cd"

// API Key（在令牌管理中创建）
api_key: "sk-your-token-here"

// 分组选择（根据需求选择）
group: "standard" // 或 "premium", "enterprise"

配置完成后，即可在您的智能体中享受 MaSiKe-ai 的所有模型服务！

支持的智能体与框架

  • Codex - AI 代码助手框架
  • NewAPI - 聚合 API 网关
  • NextChat - 跨平台聊天客户端
  • LangChain - LLM 应用开发框架
  • Flowise - 可视化 AI 工作流构建器

私有化部署 (LangBot)

若您需要一键搭建个人专属 AI 站点，可利用平台提供的 Docker 镜像一键运行私有服务器。
