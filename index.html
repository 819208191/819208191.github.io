1. 注册与登录系统

欢迎使用 MaSiKe-ai。为了保障您的数字资产安全，请首先完成账号的注册与登录。

  • 访问官网首页，点击右上角的 「注册」 按钮。
  • 按照提示输入您的邮箱账号并设置密码。
[ 图 1.1 - 账号注册与登录界面 ]

2. 进入控制台 (Dashboard)

成功登录后，点击顶部导航栏的 「控制台」，进入您的个人专属管理中枢。

  • 在这里，您可以直观地看到实时的 可用余额 与 调用统计，彻底告别余额焦虑。
  • 左侧菜单栏包含了聊天、绘图、充值等所有核心功能的入口。
[ 图 2.1 - 控制台全局数据概览 ]

3. 文本聊天与模型选择

如果您需要进行论文润色、代码编写或日常问答，请在左侧菜单点击 「聊天」 进入对话界面。

选择模型算法

点击界面左上角的模型下拉菜单，您可以自由切换当前对话使用的底层算力。我们提供多种最新模型以满足不同需求：

  • 支持目前最前沿的 GPT-5.6系列、GPT-4o、Claude 3.5 Sonnet。
  • 若追求性价比，也可选择 GPT-5.4、GPT-5.5 等费率更优惠的经典模型。
[ 图 3.1 - 文本聊天终端与下拉模型矩阵选择 ]

  • 我们接入了当前最先进的图像生成模型（如 image-2），成图质量极高。
  • 平均每张高清图仅花费约 0.3-0.4 元，极具性价比。

在 MaSiKe-ai，您的数据隐私是我们的最高准则。我们采用物理级+逻辑级双重隔离机制：

🔒 100% 对话隐私隔离

每个用户的对话线程和 API 密钥都是完全独立的。系统后台强制执行阅后即焚策略，您看不见其他人的聊天内容，别人（包括平台管理员）也绝对无法查看您的历史数据与核心机密。 请放心用于科研数据处理与企业级商业开发。

6. 支付与充值流程

为了满足不同使用频次的用户，平台提供两种高度灵活的充值方式：

方式一：订阅充值固定套餐

适合日常对话较多的用户，一次购买，周期内享有固定配额，网页端直接免密调用，省心省力。

方式二：余额直充 (按量计费 & 倍率分组)

极客与开发者的首选。将资金充入账户余额，用多少扣多少。在创建 Token 令牌时，您可以根据项目需求选择不同的底层模型分组，享受不同的计费倍率：

  • 特惠测试组： 低倍率，适合高频大批量基础文本任务。
  • 标准商用组： 1:1 官方倍率，保障响应速度与稳定性。
