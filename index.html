05 / 能源注入：双轨充值体系

为了满足不同体量的算力需求，MaSiKe-ai 提供高度灵活的两种充值与消费模式，请在左侧菜单栏选择 【钱包管理】 补充您的数字资产：

模式一：订阅制固定套餐（省心推荐）

适合轻度高频的 C 端用户（如日常办公、学生）。

  • 按月支付固定费用，解锁网页端直连对话权限。
  • 无需关心底层 Token 消耗算法。
  • 注意： 高峰期（晚 20:00-24:00）可能触发算力池限流，排队属正常现象。重度开发与商用强烈推荐使用模式二。

模式二：按量计费余额直充（API 开发者/重度首选）

直接往账号余额中注入资金，调用 API 时精确到每一个 Token 扣费，用多少扣多少，绝不浪费。

灵活分组与倍率控制：
当您在后台创建令牌（Token）时，可自由选择底层的模型池分组。不同分组享有不同的通道质量与计费倍率：
- 基础折扣池 (倍率 0.5x)：适合日常测试与大批量低要求文本生成，性价比极高。
- 商用标准池 (倍率 1.0x)：官方直连质量，低延迟，适合生产环境核心应用。
- 企业专线池 (倍率 2.5x)：独占高可用带宽，100% SLA 保障，无惧任何晚高峰。
